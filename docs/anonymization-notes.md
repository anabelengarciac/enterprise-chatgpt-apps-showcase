# Anonymization Notes

This repository is a portfolio case study, not a source-code release.

## Excluded

The following have been intentionally excluded:

- Company source code
- Internal repository names
- Internal URLs, domains, endpoints, and email addresses
- API tokens, credentials, environment variables, and secrets
- Proprietary datasets
- Screenshots of internal tools
- Internal brand lists, record IDs, access groups, or configuration details
- Implementation details that could reveal infrastructure or security design

## Generalized

The following have been generalized:

- Specific app names
- Vendor and platform references where not required for understanding the work
- Internal workflow names
- Technical ownership boundaries
- Submission destinations for feedback
- Exact connector configuration

## Preserved

The repository preserves:

- The type of project
- The business value
- The product and technical concepts
- The nature of my contribution
- The skills demonstrated across discovery, testing, iteration, and stakeholder alignment

## Purpose

The goal is to show professional impact in a way that is useful for recruiters and hiring managers while protecting company confidentiality.
